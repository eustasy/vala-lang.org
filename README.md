## Installation
- PHP 5.6 or 7.0
- Run `git submodule update --recursive`

## Contribution
- Expect _everything_ to be HTTPS.
- `_folders` that start with underscores are hidden on the server.
- `hooks/latest_version.php` generates `_cache/latest_version.php`
